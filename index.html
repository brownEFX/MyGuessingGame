Guessing Game

1. Add an if statement which checks if the guess is correct.
2. If it is, then display a message: Correct Guess!, otherwise “Sorry, incorrect Guess!
3. How can you combine line 2 and 3 into one line.
4. Expand the program to do this in a while loop, only exiting when the guess was correct.
5. While the guess is incorrect, test also if it is too low or too high and display the message “Incorrect, too low” or “Incorrect, too high”.
6. When you are done make the secret number random.
